# Web Tracking Analyzer

Drive a fully rendered website through a scripted sequence of user actions, capture all network traffic and WebSocket frames generated by those actions, and produce a report explaining what tracking data was sent.

See [ARCHITECTURE.md](ARCHITECTURE.md) for design details and the development roadmap.

## Quickstart

```bash
# One-time setup
uv venv
source .venv/bin/activate
uv pip install -e .
playwright install chromium

# Run a session
tracker run scripts/redoio-home.yaml

# Output appears under runs/<timestamp>_<session-name>/
```
